```text
на данный момент у меня есть файл с тренировками гармин выгруженный за год. я хочу сделать ui аналитику из файла в папке в корне проекта папка называется from_garmin_official_export. в статистике сейчас должна быть отображен только один график - кривая. гранулярность кривой - одна неделя строго от понедельника до воскресенья. данная кривая показывает усердие в тренировках рассчитаное по формуле: кол-во тренировок

над кривой - кол-во недель участвующих в статистике - это всегда все доступные недели без возможности уменьшения кол-ва доступного в статистике, то есть кол-во недель отображает доступное кол-во недель в статистике.

еще выше кол-ва недель - мое имя "Илья"
```
